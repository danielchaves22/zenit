# Zenit Calc frontend

## Objetivo

Registrar a direcao do frontend dedicado do Zenit Calc, consumindo a mesma API multi-tenant do ecossistema Zenit.

## Escopo

O frontend do Zenit Calc existe para atender clientes que exigem UX, branding e autenticacao dedicados sem duplicar backend e banco.

## Publico-alvo

- produto;
- desenvolvimento;
- lideranca tecnica envolvida na evolucao do Zenit Calc.

## Conceitos principais

- frontend proprio para a experiencia do Zenit Calc;
- autenticacao e navegacao isoladas;
- reaproveitamento de contratos e componentes compartilhados sempre que fizer sentido;
- validacao de habilitacao do usuario e da empresa antes da liberacao das rotas do produto.
